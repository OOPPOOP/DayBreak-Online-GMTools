存取 UDDI API 及 Microsoft API 擴充程式 介面

UDDI 服務支援 UDDI 1.0 及 2.0 版的 API 規格，以及用於瀏覽類別化配置的擴充程式 API。

建議您使用 Microsoft® UDDI .NET Software Development Kit (SDK) 來存取這些介面，因為它在使用直接從 Web Service Description Language (WSDL) 檔案產生的 Proxy 方面提供絕佳的程式設計經驗。若要下載 SDK 及存取其他 UDDI 資訊，請參閱 Microsoft 網站上的 UDDI Services 網站。

  • UDDI WSDL 檔案可從 UDDI specification 網站上取得。
  • Microsoft API 擴充程式 WSDL 檔案可從 Microsoft UDDI Business Registry 網站上取得。
