访问 UDDI API 和 Microsoft API 扩展接口

UDDI 服务支持 UDDI 1.0 和 2.0 版本的 API 规范，以及扩展 API 以浏览分类方案。

建议使用 Microsoft® UDDI .NET 软件开发工具包 (SDK) 来访问这些接口，因为与使用直接从 Web 服务描述语言 (WSDL) 文件生成的代理相比，它能提供优越的编程体验。要下载 SDK 和访问其他 UDDI 信息，请参阅 Microsoft 网站上的 UDDI 服务网站。

  • 可从 UDDI 规范网站上获得 UDDI WSDL 文件。
  • 可从 Microsoft UDDI 业务注册表网站上获得 Microsoft API 扩展 WSDL 文件。
